git init . -> cria um repo git na pastal atual
git status -> mostra o status do trabalho (de cada arq)
git add nome_arquivo -> adiciona arquivo para ser commitado
git commit -m "msg" -> faz commit do que estava para ser commitado
git logo -> mostra todo o histórico de commit

## Fluxo de trabalho

1. criar o diretório do projeto
2. iniciar o git (git init .) 
3. edita ou cria arquivos
4. git status
5. git add (nome arquivos)
6. git status
7. git commit -m "mensagem"
8. git status
9. Volta para para o passo 3
